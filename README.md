# Procloud_AI_Factory

## Frontend Prototype

The clickable AI Factory prototype lives in `frontend/`.

```bash
cd frontend
npm install
npm run dev
```

Open `http://localhost:3000`.

Useful checks:

```bash
cd frontend
npm run test
npm run build
npm run e2e
```

The prototype uses static data only. It does not call Plane, FastAPI, LangGraph, SCM, or real agents.
